# AI Fake GPS for Rooted Android

Ứng dụng giả lập GPS thông minh cho thiết bị Android đã root bằng Magisk.

## Tính năng
- Giả lập vị trí GPS
- Hỗ trợ thiết bị đã root
- Giao diện đơn giản, dễ sử dụng

## Cài đặt
1. Clone repository
2. Mở bằng Android Studio
3. Build và cài đặt trên thiết bị

## Cảnh báo
Chỉ sử dụng cho mục đích hợp pháp như phát triển và kiểm thử ứng dụng.
